Crafti.ca » donations We Build the World Mon, 21 Oct 2013 17:47:09 +0000 en-US hourly 1 http://wordpress.org/?v=3.8 Command changes and more. /command-changes-and-more/ /command-changes-and-more/#comments Mon, 01 Apr 2013 15:20:11 +0000 /?p=747 In response to the comments I have been receiving concerning the permissions on the server, the following changes have been made:

Guests now have /tpa and /tpahere in order to more easily navigate the server and work together. Additionally, the /sethome, /delhome, and /home commands are now available for guest use. Guests are reminded that they can always return to their claimed plots in Elysium with a simple /plot home command.

Builders have access to the /mvtp command to traverse between worlds faster without first having to return to The Crossroads. Usage is as simple as typing /mvtp [worldname], though please note that the world name must be fully typed out, as Multiverse does not infer your destination for you. A full list of available worlds can be obtained with the /mv list command.

Because some of these features were previously available only as donator perks, the donator rewards have accordingly changed and some perks have been rearranged.

  • Tier 1 donators gain the ability to use the /hat command to use blocks as festive head ware, and now have access to the /thru, /ascend, and /descend commands and the ability to use the compass item for these same powers; left clicking will teleport you to the area you are looking at, while right clicking will go through it.
  • Tier 2 donators now have access to the WorldEdit //repl tool, which allows for easy block substitutions with just a click.
  • Tier 3 donators gain the /back command to return to a location they were in before they teleported or crossed worlds.
  • Tier 4 donators now have the ability to use /lightning and /fireball for an extra bit of fun.
]]> /command-changes-and-more/feed/ 0 Crafti.ca is nearing launch! /crafti-ca-is-nearing-launch/ /crafti-ca-is-nearing-launch/#comments Mon, 18 Feb 2013 00:54:13 +0000 /?p=550 We’ve been hard at work setting up the foundation of Crafti.ca – finishing the Crossroads, tweaking plugins and permissions, and ensuring that all the bells and whistles we’re implementing work together as seamlessly as possible.

Currently, we’ve managed to set the donations page up to interface with the server directly, so any promotions or perks associated with donating should take place almost immediately. We’ve also set up a handy little monthly donation bar on the right-hand side of the site, so you can see just how your contributions help the server meet its goals! We’re currently hosted via the fantastic guys over at EnforcerCraft, who offer both dirt-cheap hosting and powerful options for us to hopefully grow into down the road.

A copy of the code powering Minotar has also been implemented on the site, and it follows a similar URL structure. We’ve successfully integrated this into both the main site and our forums, so any time an avatar is required it should automatically default to your minecraft skin, provided the username you use is the same as your in-game Minecraft name. Similarly, the server status checks have now successfully been implemented throughout the whole site, and should provide at-a-glance information fairly reliably in the top right of every page in the header.

We’ve also been looking at pushing the same sort of permissions for donators on the forum in the form of fancy forum badges, but thusfar we have been unsuccessful in actually making the server and phpBB communicate effectively. There are a few bukkit plugins that already exist to solve this problem, but all of them are designed to pull information and permissions from the forums to the server, and not to push them the other way around. Also complicating this task is the fact that phpBB forums are very security-minded, and as a result are incredibly paranoid about accepting any inbound data from outside sources.

While Ein furiously tries to finish the back-end of things and rewrite site content, those of you who have gotten early access are encouraged to pursue the job of turning the Crossroads into something more resembling an adventure map for our future exploration-minded guests to enjoy. At this point, we anticipate launch is only a few days away – stay tuned!
